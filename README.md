# R CPC engine
- This application aim to provide various clinical services utilizing the exsisting, recognized models into the real world.

- Features
1. Web-based (R-shiny)
2. Runnig nlme models for clinical pharmacokinetic consultation service, Korea
3. Needed libraries
   - shiny
   - shinydashboard
   - dashboardthemes
   - shinyWidgets
   - devtools
   - rsconnect
   - RxODE
   - nlmixr
   - reticulate
   - dplyr
   - ggplot2
   - plotly
   - rhandsontable
